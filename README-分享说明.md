# 小柯桌面萌宠 · 分享说明

一款柯南风 Q 版桌面悬浮萌宠：平时安静待在桌面角落陪你，随时间自动“饿/渴/脏/心情/精力”变化；点一下它弹出互动菜单（摸摸、喂食、喝水、玩耍、洗澡、商店、打工、睡觉），还能开机自启动。

---

## 一、怎么发给朋友（最简单）

把 **`小柯桌面萌宠-v5.0-便携版.zip`** 发给你想分享的人即可（微信文件传输助手、QQ、网盘都行）。

朋友收到后：
1. 解压 zip（**注意：整文件夹一起解压，不要只拖出 exe**），会得到一个 `win-unpacked` 文件夹。
2. 进文件夹，双击 **`小柯桌面萌宠.exe`**。
3. 小柯就出现在桌面角落啦，默认开机自动启动。

> 无需安装任何环境（不用装 Node.js / Electron），Windows 7 及以上基本都能直接跑。
> 如遇杀毒软件误报，选择“允许/信任”即可——这是未签名程序的正常现象。

## 二、基本玩法

- **单击小柯**：弹出互动菜单（摸摸 / 喂食 / 喝水 / 玩耍 / 洗澡 / 商店 / 打工 / 睡觉）。
- **双击小柯**：隐藏 / 显示（藏起来时它在系统托盘里）。
- **右下角托盘图标**：右键可“显示/隐藏”、勾选“开机自启动”、或“退出”。
- **五项状态**：饱食、口渴、清洁、心情、精力会随真实时间自然变化（离线也会结算），低于阈值小柯会表现出对应表情。
- **打工小游戏**：点“打工”赚金币，再用金币在“商店”买吃喝玩。

## 四、常见问题

- **解压后打不开？** 确认是把整个 zip 解压成文件夹，再进文件夹双击 exe；不要直接从压缩软件里双击。
- **被杀软拦了？** 未签名的 exe 常被误报，加白名单/允许运行即可。
- **想关掉开机自启？** 右键托盘图标 → 取消勾选“开机自启动”。
- **想彻底退出？** 右键托盘图标 → “退出”。

## 五、自动更新已开通（GitHub Releases）

软件已内置自动更新（electron-updater + `app-update.yml` 指向 GitHub）。仓库已建好（`github.com/liangbai666/xiaoke-desktop-pet`），v4.0.1 已发布，**朋友那边打开软件就会自动检查更新**，不用再手动发压缩包。

### 你以后怎么发新版（只需这一步）
**双击项目里的 `publish.bat`** 即可，脚本会自动：
1. 版本号自动 +1（如 4.0.1 → 4.0.2）；
2. 构建并把新 exe 上传到 GitHub Releases（草稿会自动发布）；
3. 自动生成并上传 `latest.yml`（自动更新必需）。

> 注意：`publish.bat` 里已写好了你的 GitHub 令牌，且**已被 git 忽略、不会上传到仓库**，请勿把它分享给别人或提交到公开仓库。

之后朋友那边：
- 已装过的，下次打开自动检查 → 后台下载 → 托盘弹“重启并更新”→ 点一下即升级。
- 新朋友照第一节拿压缩包解压即可，之后的更新同样全自动。

## 六、开发者 / 重新打包（可选）

源码在本项目目录：
- `index.html` + `images.js`：界面与逻辑（`images.js` 含全部图片，必须与 `index.html` 放在同一文件夹）。
- `main.js` / `preload.js`：Electron 悬浮窗、托盘、开机自启、自动更新。
- `app-update.yml`：自动更新配置（已通过 `extraResources` 内嵌进打包产物）。
- `bump-version.js` / `upload-latest-yml.js`：发版辅助（版本自增、生成并上传 latest.yml）。
- `publish.bat`：一键发版（含令牌，**本地保留，勿外传**）。
- 仅本地打包：`npm install` → `npm run dist`，产物在 `dist/`。
- 发版到 GitHub（含自动更新）：双击 `publish.bat`。
