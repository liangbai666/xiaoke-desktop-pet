# 小柯桌面萌宠 · 分享说明

一款柯南风 Q 版桌面悬浮萌宠：平时安静待在桌面角落陪你，随时间自动“饿/渴/脏/心情/精力”变化；点一下它弹出互动菜单（摸摸、喂食、喝水、玩耍、洗澡、商店、打工、睡觉），还能开机自启动。

---

## 一、怎么发给朋友（最简单）

把 **`小柯桌面萌宠-v4.0-便携版.zip`** 发给你想分享的人即可（微信文件传输助手、QQ、网盘都行）。

朋友收到后：
1. 解压 zip（**注意：整文件夹一起解压，不要只拖出 exe**），会得到一个 `win-unpacked` 文件夹。
2. 进文件夹，双击 **`小柯桌面萌宠.exe`**。
3. 小柯就出现在桌面角落啦，默认开机自动启动。

> 无需安装任何环境（不用装 Node.js / Electron），Windows 7 及以上基本都能直接跑。
> 如遇杀毒软件误报，选择“允许/信任”即可——这是未签名程序的正常现象。

---

## 二、基本玩法

- **单击小柯**：弹出互动菜单（摸摸 / 喂食 / 喝水 / 玩耍 / 洗澡 / 商店 / 打工 / 睡觉）。
- **双击小柯**：隐藏 / 显示（藏起来时它在系统托盘里）。
- **右下角托盘图标**：右键可“显示/隐藏”、勾选“开机自启动”、或“退出”。
- **五项状态**：饱食、口渴、清洁、心情、精力会随真实时间自然变化（离线也会结算），低于阈值小柯会表现出对应表情。
- **打工小游戏**：点“打工”赚金币，再用金币在“商店”买吃喝玩。

## 四、常见问题

- **解压后打不开？** 确认是把整个 zip 解压成文件夹，再进文件夹双击 exe；不要直接从压缩软件里双击。
- **被杀软拦了？** 未签名的 exe 常被误报，加白名单/允许运行即可。
- **想关掉开机自启？** 右键托盘图标 → 取消勾选“开机自启动”。
- **想彻底退出？** 右键托盘图标 → “退出”。

## 五、开启自动更新（GitHub Releases，推荐）

软件已内置自动更新（electron-updater）。以后你发新版，朋友那边启动时就会自动下载安装，不用再手动发压缩包。

你只需做一次、且唯一要手动的，是去 GitHub 生成一个“令牌”（Token）——因为仓库和令牌都绑定在你的账号下，这一步必须由你本人操作（涉及账号密码/验证码，任何人都不能代劳）。其余全部自动化。

### 第 1 步：生成 GitHub 令牌（一次性）
1. 打开 github.com 并登录（没有账号先注册，免费）。
2. 右上角头像 → Settings → 左侧 Developer settings → Personal access tokens → Tokens (classic)。
3. 点 **Generate new token (classic)**，随便写个名字（如 xiaoke），Expiration 选 No expiration（或随便选）。
4. 在权限里勾选 **repo**（这一整段都勾上）。
5. 拉到底点 **Generate token**，**把生成的一长串字符复制保存好**（只显示一次）。

### 第 2 步：一键发布（以后每次更新都只跑这一个）
双击项目里的 **`publish.bat`**，把刚才复制的令牌粘贴进去回车。脚本会自动：
- 用令牌拿到你的用户名、改好配置；
- 创建公开仓库 `xiaoke-desktop-pet`（已存在则跳过）；
- 构建并把新版本上传到你的 GitHub Releases。

之后朋友那边：
- 装过旧版 exe 的，下次打开自动检查 → 后台下载 → 托盘弹“重启并更新”→ 点一下即升级。
- 新朋友照第一节拿压缩包解压即可，之后的更新同样全自动。

> 说明：GitHub 公开仓库对客户端检查更新免费且不限速。若你不想用 GitHub，也可以把更新文件放到自己的服务器/对象存储（把 `build.publish` 改成 `generic` 类型并填网址）；需要我帮你改配置随时说。

## 六、开发者 / 重新打包（可选）

源码在本项目目录：
